# Installation

To install CPAC, you can download the pre-compiled binaries from the release page or build from source.

## Quick Install Script

For most systems, run:

```bash
curl -sL https://thecinderproject.qd.je/install.sh | bash
```

## Building from source

```bash
git clone https://github.com/sabeeirsharrma/cpac.git
cd cpac
make install
```
